- Good-practice = 1st create interface then create service class.
- jackson dependency helps in printing the json format.

- controller calling --> service  --> calling dao/repo.

- @Autowire can be done for interface , can be done for class but better to use for interface.

- Controller --> Service --> Dao (IEmployeeRepo)
